Acerca de esta aplicación

¿Qué es esta app?

Una aplicación web para guardar y acceder fácilmente a tus pases de bus. Proyecto personal creado para mi uso y el de mi entorno cercano.

Privacidad y seguridad

Tus imágenes se guardan únicamente en tu dispositivo, de forma local. Nunca se envían a internet ni a ningún servidor externo. Todo permanece en tu teléfono.

La aplicación funciona completamente offline una vez instalada.
